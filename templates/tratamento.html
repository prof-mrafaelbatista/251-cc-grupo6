Tratamento de Exceções

Vamos aprender como tratar exceções usando Python.

O que é?

O tratamento de exceções em Python permite lidar com erros que ocorrem durante a execução do código, evitando que o programa termine abruptamente.

Utiliza-se a estrutura try...exceptpara encapsular o código que pode gerar exceções e, em seguida, capturar e tratar as exceções que ocorrem.

Como funciona:

  • Try: Este bloco contém o código que pode gerar uma exceção.
  • Except: Este bloco é executado se uma exceção ocorrer dentro do bloco try. Permite especificar o tipo de exceção que se pretende capturar.
  • Finally (opcional): Este bloco é executado sempre, independentemente de ocorrer ou não uma exceção. É frequentemente utilizado para fechar arquivos ou liberar recursos.

Exemplo:

try: # Código que pode gerar uma exceção
num = 10 / 0
except ZeroDivisionError: # Código a ser executado se uma ZeroDivisionError ocorrer
print("Erro: Divisão por zero!")
except Exception as e: # Código a ser executado se outra exceção ocorrer
print(f"Ocorreu um erro: {e}")
finally: # Código que será sempre executado
print("Finalizando a execução.")

Pontos Importantes:

  • O bloco except pode ter vários blocos para tratar diferentes tipos de exceções.
  • Se não houver tratamento específico para uma exceção, ela será propagada para o bloco except mais externo ou, se não houver, o programa será interrompido.
  • A cláusula elsetambém pode ser utilizada após a cláusula except, sendo executada apenas se a cláusula try for executada sem gerar exceções.

Utilização na Prática

O tratamento de exceções é fundamental para tornar o código mais robusto e resiliente a erros inesperados. Permite:

  • Lidar com situações de erro de forma elegante e controlada.
  • Evitar que o programa termine abruptamente.
  • Melhorar a experiência do utilizador, exibindo mensagens de erro apropriadas.
  • Facilitar o desenvolvimento e a manutenção do código, tornando-o mais fácil de compreender e depurar.

Conclusão: o tratamento de exceções em Python é uma ferramenta essencial para construir aplicações mais seguras e confiáveis!
