Funções e Procedimentos

Entenda de uma vez por todas o conceito e aplicação com exemplos práticos!

Definição

Em Python, funções e procedimentos são mecanismos para organizar e reutilizar código.

As funções (functions) são capazes de retornar valores, enquanto os procedimentos (procedures) não.

Em Python, a palavra-chave def é usada para definir tanto funções quanto procedimentos. A diferença reside no retorno: função retorna um valor (usando a instrução return) , enquanto um procedimento não.

Funções

São caracterizadas por retornar valores, ou seja, são criadas para executar uma tarefa e, em seguida, retornar um valor com o resultado.

Esse valor pode ser usado em outras partes do seu código, evitando um retrabalho.

Exemplo:

def soma(a, b): # Função que retorna a soma de dois números.
return a + b
resultado = soma(5, 3)
print(resultado) # Imprime 8

Procedimentos

Os procedimentos são executados para realizar uma ação, mas não retornam um valor que possa ser usado em outras partes do seu código.

Permitem encapsular blocos de código que realizam ações específicas, sem a necessidade de retornar valores. 

def imprimir_mensagem(texto):    # Procedimento que imprime uma mensagem no console.
print(texto)
imprimir_mensagem("Olá, mundo!")  # Imprime "Olá, mundo!" no console

Conclusão

Em resumo, tanto funções quanto procedimentos são ferramentas essenciais para a construção de programas organizados e eficientes em Python. A escolha entre eles depende do que você precisa que o seu código realize: calcular um valor para ser usado posteriormente (função) ou executar uma ação (procedimento).
