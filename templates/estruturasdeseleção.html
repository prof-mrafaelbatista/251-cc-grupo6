Estruturas de Seleção em Python

As estruturas de seleção são fundamentais na programação, pois permitem que o código tome decisões com base em condições específicas. Elas possibilitam alterar o fluxo de execução do programa, fazendo com que determinados blocos de código sejam executados apenas se certas condições forem verdadeiras.

Quando usar estruturas de seleção?

Você deve utilizar estruturas de seleção sempre que houver a necessidade de que o programa se comporte de maneira diferente dependendo de determinadas situações, como:

  • Verificar se um usuário tem idade suficiente para acessar um serviço;
  • Checar se um número é positivo, negativo ou zero;
  • Determinar qual desconto aplicar com base no valor da compra;
  • Tomar decisões baseadas em entradas do usuário.

A estrutura if

A estrutura if é utilizada para executar um bloco de código apenas se uma determinada condição for verdadeira. Caso a condição seja falsa, o bloco não será executado.

Exemplo:


numero = int(input("Digite um número: "))

if numero > 0:
    print("O número é positivo.")

Neste exemplo, a mensagem será exibida apenas se a variável numero for maior que zero.

A estrutura else

A estrutura else complementa o if, permitindo que um bloco de código seja executado quando a condição do if for falsa. Ou seja, quando o if não é satifeito, o else entra em ação.

Exemplo:


numero = int(input("Digite um número: "))

if numero > 0:
    print("O número é positivo.")
else:
    print("O número é zero ou negativo.")

Aqui, se o número não for maior que zero, o programa executa automaticamente o bloco dentro do else.

A estrutura elif

O elif (abreviação de "else if") é utilizado quando precisamos testar múltiplas condições. Ele permite adicionar várias verificações, cada uma com seu próprio bloco de código.

Exemplo:


numero = int(input("Digite um número: "))

if numero > 0:
    print("O número é positivo.")
elif numero == 0:
    print("O número é zero.")
else:
    print("O número é negativo.")

Neste exemplo, o programa verifica inicialmente se o número é maior que zero. Se não for, ele verifica se é igual a zero. Caso nenhuma das condições anteriores seja satisfeita, o else é executado.

Resumo das Estruturas:

  • if: executa um bloco se a condição for verdadeira.
  • elif: adiciona novas condições a serem verificadas caso a anterior seja falsa.
  • else: executa um bloco se todas as condições anteriores forem falsas.

Exemplo completo:


nota = float(input("Digite a nota do aluno: "))

if nota >= 7:
    print("Aluno aprovado.")
elif nota >= 5:
    print("Aluno em recuperação.")
else:
    print("Aluno reprovado.")

Neste exemplo, o programa avalia a nota do aluno e imprime a mensagem adequada dependendo da situação.

Considerações finais:

As estruturas de seleção são essenciais para criar programas mais dinâmicos e inteligentes. Dominar o uso de if, elif e else é o primeiro passo para aprender a linguagem pyhton. E para aqueles que estao no seu primeiro contato com o mundo de programação, aprender a logica de pyhton, é fundamental para o desenvolvimento da forma de pensar na resolução de problemas e criação de algoritmos.
