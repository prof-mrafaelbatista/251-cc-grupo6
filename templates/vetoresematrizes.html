Vetores e Matrizes

Entenda de uma vez por todas o conceito e aplicação com exemplos práticos!

Vetores: Lista Unidimensional

É uma sequência ordenada de elementos. Em Python, isso é representado por uma lista com uma única dimensão.

Exemplo de vetor:

nomes = ["Ana", "Carlos", "João"]
print(nomes[0])  # Saída: Ana

No Python, o primeiro elemento de um vetor tem índice 0. Então nomes[0] representa “Ana”, nomes[1] “Carlos” e assim por diante.

Outro exemplo:

idade = ["11", "12", "13"]
print(idade[2])  # Saída: 13

idade[0] representa "11", idade[1] é "12" e idade[2] é "13".

Para adicionar um novo item usamos append:

idade = ["11", "12", "13"]
idade.append("14")
print(idade)

A saída agora incluirá o elemento "14".

Matrizes: Lista Bidimensional

Uma matriz é uma estrutura com linhas e colunas. Em Python, é uma lista de listas.

Exemplo:

matriz = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

Para acessar o elemento na 2ª linha e 3ª coluna:

print(matriz[1][2])  # Saída: 6

Como o índice começa em 0, matriz[1][2] corresponde à 2ª linha e 3ª coluna.

print(matriz[0][0])  # 1 → primeira linha, primeira coluna
print(matriz[1][2])  # 6 → segunda linha, terceira coluna
print(matriz[2][1])  # 8 → terceira linha, segunda coluna

Como utilizar Vetores e Matrizes na Prática

Vetores são listas simples e matrizes são listas de listas. Vamos ver isso funcionando:

vetor1 = [1, 2, 3]
vetor2 = [4, 5, 6]
vetor3 = [7, 8, 9]

Agora combinando esses vetores em uma matriz:

matriz = [
  [1, 2, 3], # Vetor 1
  [4, 5, 6], # Vetor 2
  [7, 8, 9]  # Vetor 3
]

Temos uma matriz 3x3! Super fácil, certo? 😄
