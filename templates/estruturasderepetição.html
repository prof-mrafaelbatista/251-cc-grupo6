Estruturas de Repetição em Python

As estruturas de repetição, também chamadas de loops, são usadas para executar um bloco de código várias vezes, de acordo com uma condição ou para percorrer elementos de uma sequência (como listas, strings, dicionários, etc.).

Exemplo:

frutas = ['maçã', 'banana', 'uva']
            for fruta in frutas:
            print(fruta)

Em Python, as estruturas de repetição são utilizadas para executar um bloco de código várias vezes. As principais estruturas de repetição em Python são for e while. O for é usado quando se sabe o número exato de repetições ou quando se precisa iterar sobre um iterável, como uma lista ou string. O while é usado quando a repetição depende de uma condição lógica, que deve ser verdadeira para o loop continuar.


Sintaxe:

for variável_de_controle in objeto_iterável:
variável_de_controle: Recebe o valor de cada elemento do objeto iterável. objeto_iterável: Pode ser uma lista, tupla, string, conjunto, etc.
